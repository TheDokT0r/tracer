# Keybindings

## Global

| Key            | Action               |
|----------------|----------------------|
| `tab`          | Next tab             |
| `shift+tab`    | Previous tab         |
| `:`            | Command palette      |
| `q` / `ctrl+c` | Quit                |

## Session List

| Key     | Action                 |
|---------|------------------------|
| `up`/`down` | Navigate           |
| `enter` | Resume session         |
| `v`     | View detail            |
| `n`     | New session            |
| `f`     | Fork session           |
| `c`     | Copy session ID        |
| `p`     | Pin/unpin              |
| `d`     | Delete                 |
| `/`     | Filter                 |
| `s`     | Settings               |

## Session Detail

| Key     | Action            |
|---------|-------------------|
| `up`/`down` | Scroll       |
| `enter` | Resume            |
| `f`     | Fork              |
| `r`     | Rename            |
| `e`     | Edit in $EDITOR   |
| `x`     | Export            |
| `c`     | Copy ID           |
| `d`     | Delete            |
| `esc`   | Back              |

## Skills List

| Key       | Action          |
|-----------|-----------------|
| `up`/`down` | Navigate     |
| `enter`/`v` | View detail |
| `e`       | Edit            |
| `n`       | New skill       |
| `d`       | Delete          |
| `/`       | Filter          |

## Skills Detail

| Key   | Action          |
|-------|-----------------|
| `e`   | Edit            |
| `d`   | Delete          |
| `esc` | Back            |

## Permissions List

| Key       | Action          |
|-----------|-----------------|
| `up`/`down` | Navigate     |
| `enter`/`v` | View rules  |
| `/`       | Filter          |

## Permissions Detail

| Key   | Action                   |
|-------|--------------------------|
| `a`   | Add rule                 |
| `t`   | Toggle allow/deny        |
| `d`   | Delete rule              |
| `esc` | Back                     |

## Settings

| Key                  | Action       |
|----------------------|--------------|
| `up`/`down` or `k`/`j` | Navigate  |
| `left`/`right` or `h`/`l` or `enter` | Change value |
| `esc` / `q`         | Save & exit  |

## Command Palette

| Key         | Action           |
|-------------|------------------|
| `enter`     | Execute          |
| `tab`       | Accept suggestion |
| `up`/`down` | Navigate / history |
| `esc`       | Cancel           |
